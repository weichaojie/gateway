# Spring Cloud Gateway 工程说明

- `gateway-basic`
  - 该工程是基本入门工程，提供了相关Demo
- `gateway-customizer`
  - 该工程是自定义路由断言和路由过滤器工程
- `gateway-limiter`
  - 该工程是网关限流工程
- `gateway-danymic-route`
  - 该工程是动态路由工程
- `gateway-swagger`
  - 该工程整合swagger，用于展示位于gateway后的服务文档信息
- `gateway-logging`
  - 该工程是gateway同一格式输出请求响应日志工程
